# Azure DevOps AI Assistant

Professional Node.js tool to connect to Azure DevOps and generate
AI-powered test cases from Tasks / Work Items.

## Setup
npm install
npm start

## Features
- Azure DevOps integration (PAT)
- AI-generated test cases
- Scalable architecture
- Enterprise-ready

## Extend
- Acceptance Criteria (Gherkin)
- Risk analysis
- Test Plan creation
- CI/CD integration

## 🚀 Cómo ejecutar
En la carpeta raíz:
npm install
npm start
